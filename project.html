project & technologies

We run several projects mostly in the area of research, software engineering and advocacy.
Early 2012 we developed a Project Plan that summarize our mid-term vision for most project implementation.
We aim to start-up each project we work on, in a any that would facilitate creation of an independent community around it.

All projects started on a voluntary basis by Hermes members, Fellows or spot contributors, but was able to receive support by several organizations:

GlobaLeaks 0.1 has been financed on a project by USAID Serbia.

GlobaLeaks 0.2 and Tor2web 3.0 Projects are currently financed by Freedom2Connect Program of Radio Free Asia.

Apaf Project has been supported by Google Summer of Code.

Below a summary of our currently active projects:

GlobaLeaks

GlobaLeaks is the first open-source whistleblowing framework. It empowers anyone to easily set up and maintain an anonymous whistleblowing platform. GlobaLeaks can help many different types of users: media organizations, activist groups, corporations and public agencies.

» Read more

Tor2web

Tor2web is the software project to create the Tor2web Network, a worldwide distributed network of proxy that let anyone Internet publish a website without being completely anonymous thanks to the use of Tor Hidden Services.

» Read more

LeakDirectory

Currently leakdirectory.org is a community wiki that contains the list of all whistleblowing sites useful information on the topic of whistleblowing. It will became a site (and software) providing an index of GlobaLeaks Whistleblowing sites, with their capabilities, goals, references and information. It will be the AppStore of whistleblowing sites.

» Read more

Fax2social

Fax2social is a tool to act as a bridge between analog telephony FAX and internet, allowing submission via Fax, both on public social network (Twitter, Dropbox), or on a GlobaLeaks site. It has born as anti internet-kill-switch tool following Egyptian disconnections.

» Read more

Big Brother Awards

Desc: Yearly the Big Brother Awards is presented to government agencies, private companies and individuals who have excelled in the violation of citizen privacy. The italian edition is run by PWS in cooperation with Hermes.

» Read more

Apaf

APAF, Anonymous Python Application Framework, is a technology to facilitate development of cross-platform anonymous applications, for multiple desktop and server environment. It is used to released GlobaLeaks on Windows, OSX and Linux.

» Read more

GLDroid

GLDroid is an Android GlobaLeaks Application to Submit files, Photo and Videos with integrated anonymity (Tor) and Face obfuscation (Obscuracam).
